# 📊 Employees Dashboard (Power BI)

## 🚀 Project Overview

This project is an interactive Power BI dashboard that provides insights into employee distribution, salary trends, and department-wise analysis.

## 📌 Features

* Employees distribution by center
* Department-wise employee count
* Gender-based analysis
* Job rate and salary insights
* Year-wise hiring trends

## 🛠️ Tools Used

* Power BI
* Excel / CSV Dataset

## 📊 Key Insights

* Majority employees are in Manufacturing department
* Male employees dominate workforce (~65%)
* Hiring peaked in 2019
* Average salary is around 24.8K
